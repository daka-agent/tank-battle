# 🎮 经典坦克大战

经典 FC 坦克大战游戏的 Web 重制版，纯 HTML + CSS + JavaScript 实现，零依赖，单文件即玩。

## 🎯 游戏特性

- 🟢 **玩家1**：方向键移动 + 空格射击
- 🔵 **玩家2**：WASD 移动 + F 射击（双人模式）
- 🔴 **敌方坦克**：AI 自动移动与攻击，偏向朝玩家和司令部方向
- 🧱 **可破坏砖墙** + 🔩 **不可破坏钢墙**
- 🦅 **司令部保护**：底部中央，被击中则游戏结束
- 🏆 **积分系统**：击杀 +100 分，过关 +500 分
- 📈 **关卡递进**：3 种地图布局循环，敌人数量/速度/射速逐关提升
- ❤️ **3 条命/人**，重生后 2 秒无敌闪烁
- 🎵 **完整音效**：8-bit 背景音乐、射击、爆炸、打墙、钢墙、过关、游戏结束等音效
- ⏸ **暂停/静音/重启**：P 暂停、M 静音、R 重新开始

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可游玩，无需安装任何依赖或启动服务器。

```bash
# 克隆仓库
git clone https://github.com/你的用户名/坦克大战.git

# 打开游戏
# 双击 index.html 或用浏览器直接打开
```

## 🎮 操作方式

| 操作 | 玩家1 (绿色) | 玩家2 (蓝色) |
|------|-------------|-------------|
| 上 | ↑ | W |
| 下 | ↓ | S |
| 左 | ← | A |
| 右 | → | D |
| 射击 | 空格 | F |
| 暂停 | P | P |
| 静音 | M | M |
| 重新开始 | R | R |

## 🛠 技术实现

- **Canvas 2D** 渲染，624×624 像素（26×26 格，每格 24px）
- 坦克尺寸 48×48px（占 2 格），带履带动画
- `requestAnimationFrame` 驱动游戏主循环
- **Web Audio API** 程序化生成所有音效，无外部音频文件
- 单文件架构，零外部依赖
